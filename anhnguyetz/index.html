Gửi gắm thông điệp vào dải ngân hà

Cách tạo dải ngân hà của riêng bạn:

  • Ảnh tinh cầu: Upload ảnh từ máy hoặc dán link ảnh trực tiếp. (Nên dùng 5-10 ảnh để mượt nhất).
  • Lời nhắn gửi: Nhập các đoạn văn bản sẽ bay quanh hành tinh (mỗi dòng là một đoạn mới).
  • Âm nhạc: Chọn một bản nhạc nền lãng mạn từ danh sách hoặc dán link .mp3 tùy ý.
  • Hoàn tất: Nhấn "Tạo Link", sau đó bạn có thể "Rút gọn" link cho đẹp và "Tạo QR" để chia sẻ!
Ảnh trái tim

Lưu ý quan trọng: Nếu bạn chọn Upload file trực tiếp lên trên nền tảng thì những hình ảnh này sẽ được lưu trữ bằng API Mặc Định của nền tảng trong vòng 1 Tháng. Nếu bạn muốn lưu giữ vĩnh viễn hình ảnh của riêng bạn trên nền tảng đám mây, thì bạn cân nhắc thay đổi API Key của riêng mình để sử dụng.
